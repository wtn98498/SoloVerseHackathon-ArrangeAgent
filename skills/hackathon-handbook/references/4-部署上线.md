# 部署上线

目标只有一个：**让评委能点开一个公开 URL 看到你的 demo。** 黑客松提交几乎都要一个 live link，别等到最后一刻才弄。

默认路径：**GitHub（放代码）→ Vercel（自动部署，给你公开链接）**。Next.js 项目在 Vercel 上零配置。

先判断用户水平，分两条路：
- **会用 Git / 已有 GitHub 仓库** → 直接看"快速路径"。
- **没用过 GitHub / 不懂命令行** → 走"零基础保姆教程"，一步步带。

判断不准就直接问一句："你之前用过 GitHub 吗？"

---

## 快速路径（会 Git 的人）

```bash
# 项目目录里
git init
git add .
git commit -m "hackathon demo"
# 在 github.com 新建一个空仓库，拿到地址后：
git remote add origin https://github.com/你的用户名/仓库名.git
git branch -M main
git push -u origin main
```

然后去 **vercel.com** → New Project → 选 GitHub 仓库 → Import → Deploy。约 1 分钟出公开链接。

如果有环境变量（如 AI API key、Supabase key），在 Vercel 项目的 Settings → Environment Variables 里加，然后 Redeploy。

更快的法子：装了 Vercel CLI 的话，项目目录里直接 `npx vercel` 跟着提示走也能上线（甚至不经过 GitHub），但黑客松通常要交 GitHub 链接，所以还是建议推到 GitHub。

## 零基础保姆教程（没用过 GitHub）

完全没接触过 Git/GitHub 也没关系，跟着做。先解释一句概念，降低恐惧：**GitHub 是放代码的网盘，Vercel 盯着这个网盘，你一更新它就自动把网站发布出去。**

最省事的方式是**用 GitHub Desktop（图形界面，不碰命令行）**：

1. **注册 GitHub**：打开 github.com 注册一个账号（免费）。
2. **装 GitHub Desktop**：去 desktop.github.com 下载安装，用刚注册的账号登录。
3. **把项目变成仓库**：GitHub Desktop → File → Add Local Repository → 选你的项目文件夹。如果它提示这里还不是仓库，点 "create a repository" 即可。
4. **第一次提交**：左下角填一句 Summary（如 "demo"）→ 点 "Commit to main"。
5. **发布到 GitHub**：点右上角 "Publish repository"。仓库可以设为 Public 或 Private 都行（Vercel 都能连）。点确认，代码就上云了。
6. **连 Vercel**：打开 vercel.com → 用 GitHub 账号登录（Continue with GitHub）→ Add New → Project → 找到刚发布的仓库 → Import → Deploy。
7. **等约 1 分钟**，Vercel 给你一个 `https://你的项目.vercel.app` 的公开链接。这就是交给评委的链接。
8. **之后改了代码**：在 GitHub Desktop 里再 Commit → Push，Vercel 会自动重新部署，链接不变、内容自动更新。

如果有 API key 之类的密钥：在 Vercel 项目页 → Settings → Environment Variables 里填，填完去 Deployments 点最新一次的 Redeploy。**别把密钥直接写进代码推到 Public 仓库。**

命令行版（如果用户愿意学命令行）见 `references/github-cli-steps.md`。

## 部署后自检（交付前过一遍）

- [ ] 用**无痕窗口**打开那个 vercel.app 链接，确认陌生人也能访问（不是只有你登录能看）。
- [ ] 手机上打开看一眼，排版没崩。
- [ ] demo 里要用的核心功能在线上是好的（尤其是依赖环境变量的 AI 调用——本地能跑不等于线上能跑，检查 key 配没配）。
- [ ] 把链接复制到提交表单 / Devpost，再点一次确认没填错。
- [ ] 录 demo 视频时也用这个线上版本，避免"本地能跑、现场翻车"。

链接出来后，这个项目就齐活了：idea、叙事、pitch、可点的 demo、公开链接——可以去提交了。
